## 阶段一：夯实底层

渗透的本质是“不按常理出牌”，所以你必须先懂“常理”。

1. **Linux 基础：** 掌握文件系统、权限管理、SSH、常用命令。
    
    - **资源：** **OverTheWire (Bandit系列)** —— 通过游戏闯关学 Linux。
        
2. **网络协议：** 深入理解 TCP/IP、HTTP/HTTPS、DNS、子网划分。
    
    - **资源：** **B站 / YouTube (计算机网络微课堂)**。
        
3. **脚本语言（Python/Bash）：** 能够看懂并修改简单的漏洞利用脚本。
    
    - **资源：** **廖雪峰 Python 教程** 或 **FreeCodeCamp**。
        

---

## 阶段二：Web 安全攻防

Web 是目前渗透测试中岗位最多、最容易上手的领域。

1. **漏洞原理：** 学习 OWASP Top 10（SQL注入、XSS、文件包含、SSRF等）。
    
    - **必刷靶场：** **PortSwigger Academy** —— 这是 Burp Suite 官方出的，**完全免费且包含所有 WP**，是全球公认的 Web 安全巅峰级教程。
        
2. **工具使用：** 熟练使用 Burp Suite, Nmap, SQLmap, Dirsearch。
    
    - **资源：** **CTFhub** 技能树（适合国内习惯），每个点都有配套题目。
        

---

## 阶段三：系统渗透与后渗透（成本：约 85元/月）

这个阶段你需要学习如何从 Web 漏洞渗透进服务器内网。

1. **红队基础：** 提权（Privilege Escalation）、内网转发、MSF/Cobalt Strike 使用。
    
2. **打靶实战：** * **资源：** **TryHackMe (THM)** —— 购买一个月 **VIP（约 $12）**。
    
    - **操作：** 重点走完 **"Jr. Penetration Tester"** 和 **"Offensive Pentesting"** 两条学习路径。THM 的特点是它会把服务器环境部署在网页浏览器里，不需要你自己折腾。
        
3. **进阶实战：**
    
    - **资源：** **Hack The Box (HTB)**。
        
    - **操作：** 刷 "Starting Point"（免费），如果觉得好玩，再考虑 VIP。看 **IppSec** 的视频复盘，学习大牛的思路。
        

---

## 阶段四：证书与职业化（可选，成本：50美金 - 1700美金）

如果你需要证书作为敲门砖：

1. **入门首选：** **ISC2 - Certified in Cybersecurity (CC)**。目前有 **"One Million Certified"** 计划，**免培训费和考试费**（通过后仅需付 50 美元会费），非常超值。
    
2. **实战金标：** **OSCP**。非常贵（约 $1600+），但如果你能刷完 HTB 的中等难度机器，这个证只是时间问题。建议工作后再让公司报销。

**一些“省钱”的小贴士：**

- **不要买昂贵的中文网课：** 渗透测试更新极快，B站上的免费资源 + 官方文档已经足够。
    
- **充分利用 Github：** 搜索 `Awesome Penetration Testing` 或 `Pentest Roadmap`，你会发现无数大佬整理好的免费工具合集。
    
- **本地环境：** 自己在电脑上装个 **VMware + Kali Linux**，这是最好的实验室。

## 1. 渗透实战与综合类（首选）

如果你想体验“黑掉一台服务器”的感觉，这两家是全球标杆。

### **[Hack The Box (HTB)](https://www.hackthebox.com/)**

- **特点：** 业界公认的“入行通行证”，环境极其接近真实企业内网。
    
- **如何找 WP：** * **IppSec (YouTube/官网)：** IppSec 的视频是渗透测试界的“教科书”，他会详细讲解每一个漏洞背后的原理。
    
    - **HackTheBox 官方博客：** 机器退役（Retired）后，官方会放出详细的官方 WP。
        
- **建议：** 从 "Starting Point" 路径开始，它是专门为新手设计的免费引导关卡。
    

### **[TryHackMe (THM)](https://tryhackme.com/)**

- **特点：** 相比 HTB 的“硬核”，THM 更像是一本带练习题的教科书，非常适合查漏补缺。
    
- **如何找 WP：** * **Medium/个人博客：** 直接在 Google 搜索 `TryHackMe [Room Name] writeup`，Medium 上有大量图文并茂的攻略。

## 2. 二进制安全 (Pwn & Reverse)

这个方向最吃基础，但也最硬核。

### **[pwnable.kr](https://pwnable.kr/)**

- **特点：** 韩国出品，题目非常经典，分为 Toddler's Bottle (新手级)、Rookies、Great Daemons 等。
    
- **如何找 WP：** * **GitHub：** 搜索 `pwnable.kr writeup`。由于题目太经典，网上的 WP 已经多到泛滥，非常适合对照学习。
    

### **[pwnable.tw](https://pwnable.tw/)**

- **特点：** 难度陡增，专为想进入国际顶级赛事的选手设计。
    
- **如何找 WP：** 题目更新较慢，建议看 **CTFtime** 上的历史 WP。
    

---

## 3. 密码学与 Web 专项

### **[CryptoHack](https://cryptohack.org/)**

- **特点：** 全球最好的现代密码学靶场，涵盖 AES、RSA、椭圆曲线。
    
- **如何找 WP：** 平台自带社区，完成一定题目后可以解锁论坛讨论。
    

### **[PortSwigger Web Security Academy](https://portswigger.net/web-security)**

- **特点：** 由 Burp Suite 官方出品，是目前**全球最权威的 Web 安全靶场**。
    
- **如何找 WP：** 每一道题的右上角都有一个 **"Lab solution"** 按钮，官方手把手教你如何利用漏洞。
    

---

## 4. 全球 WP 资源库 (神器级)

### **[CTFtime.org](https://ctftime.org/)**

- **用途：** 它是 CTF 界的“百科全书”。
    
- **如何使用：** 点击 **"Writeups"** 栏目，你可以按年份、按比赛搜索全球顶级战队（如 PPP, Nu1L, Dragon Sector）提交的原始思路。
    
- **学习建议：** 重点看那些排名靠前的战队的思路，他们通常能给出最优雅、最简练的代码脚本。
    

### **[HackTricks](https://book.hacktricks.xyz/)**

- **用途：** 这不是靶场，而是**最强的“备忘录”**。当你做靶场遇到瓶颈（比如不知道如何绕过某个 WAF）时，在这里搜索，它会告诉你所有已知的攻击姿势。

## 给你的第一步建议

如果你现在就有 Kali Linux 或者一套 Python 环境，我建议你按以下步骤开始：

1. **第一天：** 登录 **OverTheWire** 的 **Bandit** 关卡，先把 Linux 命令行练熟。
    
2. **第二天：** 去 **CTFhub** 刷完“技能树”中的 **Web 基础**部分（SQL 注入和文件上传）。
    
3. **第三天：** 尝试在 **Hack The Box** 注册一个账号，开始你的第一个 "Easy" 难度的机器。
    



在这些全球知名的靶场中，收费模式主要分为：**完全免费**、**订阅制（Freemium）**以及**一次性付费**。

以下是针对前文提到的主流平台的付费情况及价格汇总（以 2026 年常见标准为例）：

### 1. 订阅制平台（部分免费 + 会员权益）

这些平台通常可以免费注册并玩一些基础题目，但想要体验最新的机器或专业实验室，需要按月/按年付费。

- **Hack The Box (HTB)**
    
    - **免费版：** 可以玩“退役”前的部分活跃机器（Active Machines）。
        
    - **VIP ($14/月 或 $135/年)：** 可访问所有退役机器及其实验室 Writeup、个人统计数据、专属服务器（不拥挤）。
        
    - **VIP+ ($20/月)：** 拥有个人专属的靶机实例（不需要和别人抢），延迟更低。
        
    - **Academy (学院)：** 采用“学分制”，部分模块免费，高级模块（如 Active Directory）需购买学分（约 $18/月起）。
        
- **TryHackMe (THM)**
    
    - **免费版：** 约 20% 的房间（Rooms）免费开放。
        
    - **订阅会员 ($12/月 或 $105/年)：** 解锁所有学习路径、所有房间、浏览器内集成的 Kali 虚拟机、更快的部署速度。对于初学者，这个订阅的性价比最高。
        

### 2. 企业级/实验室平台（价格较高）

这些平台针对的是希望获得专业证书或极高难度实战经验的用户。

- **OffSec (原 Offensive Security / PWK)**
    
    - **价格：** 约 **$1,599** 起（包含 OSCP 考试机会、课程教材及 90 天实验室访问权限）。
        
    - **Learn One 计划：** 约 **$2,599/年**，包含一年内所有基础实验室的访问权。
        

### 3. 完全免费的平台（公益或学术性质）

这些平台大多由大学、安全团队或社区维护，**永久免费**且质量极高。

- **picoCTF：** 由卡内基梅隆大学维护，完全免费。
    
- **OverTheWire / UnderTheWire：** 经典的 Linux/Windows 命令行关卡，完全免费。
    
- **CryptoHack：** 密码学专项，完全免费。
    
- **PortSwigger Web Security Academy：** Burp Suite 官方出品，完全免费。
    
- **pwnable.kr / pwnable.tw：** Pwn 专项，完全免费。
    
- **CTFhub / 攻防世界 (XCTF)：** 国内主流平台，大部分题目免费，部分高级竞赛环境或训练营可能收费。
    
- **VulnHub：** 镜像下载站，完全免费（因为你是下载到本地跑）。
    

---

### 总结与建议：

1. **零元党首选：** **picoCTF** + **OverTheWire** + **PortSwigger Academy**。这三个组合足以让你从小白成长为具备 Web 和 Linux 基础的中手。
    
2. **性价比之选：** **TryHackMe 订阅**（约 85 元人民币/月）。它像一本书，带你系统地走过每一个知识点。
    
3. **进阶必备：** **Hack The Box VIP**（约 100 元人民币/月）。当你准备好冲击 OSCP 证书或者提升渗透实战能力时，这是必经之路。
    

**注意：** 对于你之前感兴趣的 **CTF 密码学 (Crypto)** 或 **Pwn**，目前全球顶尖的专项靶场（如 CryptoHack 和 pwnable 系列）几乎都是**完全免费**的。如果你主要想练习这两个方向，暂时不需要考虑付费问题。
