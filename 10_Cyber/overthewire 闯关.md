## Level 0: 初始登录

### 🎯 目标
连接到游戏服务器，获取进入基础环境的权限。

### 📚 知识点预习
- **SSH (Secure Shell)**: 用于加密远程登录到另一台计算机的协议。
- **端口 (Port)**: 服务器上特定服务的入口。OverTheWire 的 SSH 端口通常是 `2220`。

### 💡 引导思路
1. 使用终端工具（如 WSL, PowerShell 或 Termius）。
2. 输入 SSH 命令，格式通常为：`ssh 用户名@地址 -p 端口`。
3. 第一次连接时会提示“密钥指纹确认”，输入 `yes`。

### 🛠️ 实验过程
```bash
# 执行以下命令登录
ssh bandit0@bandit.labs.overthewire.org -p 2220
# 密码通常是: bandit0
```

## Level 0 -> Level 1: 寻找 readme

### 🎯 目标

在主目录（home directory）中找到一个名为 `readme` 的文件，并获取其中存储的密码。该密码将用于登录 `bandit1`。

### 📚 知识点预习

- **`ls` (List)**：列出当前目录下的文件和文件夹。
    
    - 常用参数 `-l` 可以查看文件的详细信息（权限、大小、修改日期）。
        
- **`cat` (Concatenate)**：将文件内容打印（输出）到终端。这是查看文本文件内容最常用的命令。
    
- **主目录 (`~`)**：登录后默认所在的文件夹。在 Linux 中，你可以通过输入 `cd ~` 或直接输入 `cd` 回到这里。
    

### 💡 引导思路

1. **确认文件存在**：首先需要看看当前目录下到底有些什么。
    
2. **查看内容**：一旦确认了 `readme` 文件的存在，使用读取命令将其内容“倒”在屏幕上。
    
3. **保存凭据**：屏幕上显示的那个长字符串就是下一关的密码。
    

### 🛠️ 关键操作记录

Bash

```
# 列出当前目录的文件
ls

# 读取 readme 文件的内容
cat readme

# 退出当前登录（准备用新密码登录下一关）
exit
```

### 🧠 经验笔记    
- **Linux 敏感性**：Linux 是区分大小写的。`readme` 和 `ReadMe` 是两个完全不同的文件。
